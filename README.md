vim
---


```bash
$ cd pathTo/dotfiles/
$ git clone https://github.com/nicolaspoulain/dotfiles.git
$ ln -s pathTo/dotfiles/vimrc ~/.vimrc
$ mkdir ~/.vim
$ git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle
```
open vim then :BundleInstall
